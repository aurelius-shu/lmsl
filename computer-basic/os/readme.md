### 分页 vs. 分段

1. 目的
   分页实现虚拟内存，获得更大逻辑地址；分段逻辑上划分独立地址，方便共享和维护
2. 透明性
   分页透明；分段显示划分
3. 地址
   分页一维；分段二维
4. 大小
   分页不可变，分段动态改变
